# Math Wizard - Kids Math Worksheet Generator

A modern, powerful, and easy-to-use math worksheet generator for kids. Built with vanilla JavaScript, it allows parents and teachers to create custom printable PDF worksheets for Addition, Subtraction, Multiplication, and Mixed practice.

- **Customizable Complexity**: Choose from 1-digit to 6-digit problems.
- **Dynamic Layouts**: Support for both Vertical (stacked) and Horizontal (line) equations.
- **Multi-term Addition**: Add up to 6 numbers in a single equation.
- **PDF Export**: Generate high-quality PDFs ready for printing.
- **Answer Keys**: Automatically generate an answer key on a separate page.
- **Progressive Web App (PWA)**: Works offline and can be installed on your device.
